# 💌 awesome-matching
2026年4月のトレンドで作成されたマッチングアプリ風ジョーク一覧。

## What is this?
2026年4月14日に[東大生と女子大学生をつなぐマッチングアプリUTopia](https://x.com/UTopia_matching/status/2043892080058216684)というアプリがリリースされました。しかしセキュリティや倫理の観点から議論が沸き起こり、話題になりました。

mimifuwaccさんが[電通大生のためのマッチングアプリを開発](https://x.com/mimifuwacc/status/2046621112566714574?s=20)し、これをきっかけに大喜利が始まりました。

このリポジトリはトレンドに乗って開発されたマッチングアプリ風ジョークをまとめたAwesome listです。
